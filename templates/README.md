# Templates

Este diretório contém templates reutilizáveis para skills, workflows e agentes.

## Estrutura

```
templates/
├── skill-template/
├── workflow-template/
├── agent-template/
└── README.md
```

## Como Usar os Templates

1. Copie o template apropriado para o diretório correspondente (skills, workflows ou agents)
2. Renomeie o diretório com o nome do seu item
3. Personalize os arquivos conforme necessário

## Templates Disponíveis

- **skill-template**: Template básico para criar uma nova skill
- **workflow-template**: Template básico para criar um novo workflow
- **agent-template**: Template básico para criar um novo agente
