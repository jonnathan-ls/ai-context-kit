# Workflows

Este diretório contém as configurações de workflows para o Antigravity.

## Estrutura

Organize seus arquivos de workflows da seguinte forma:

```
workflows/
├── workflow-name/
│   ├── config.yaml
│   ├── steps.md
│   └── README.md
└── README.md
```

## Como Adicionar um Workflow

1. Crie um novo diretório com o nome do workflow
2. Adicione a configuração e documentação dos passos
3. Descreva o workflow no arquivo README.md dentro do diretório

## Exemplos

Consulte o diretório `examples/` para ver exemplos de workflows.
