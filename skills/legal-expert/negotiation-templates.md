# Negotiation Templates - Linguagem e Estratégia

## Email Templates

### Template 1: Initial Response to Contract (Professional)

```
Assunto: Parecer sobre contrato [Nome/ID] - Questões para Discussão

Prezados [Nomes],

Agradeço por enviar o contrato. Realizei análise preliminar e identifiquei 
alguns pontos que gostaríamos de discutir antes de prosseguir com a assinatura.

Nem todas são objeções, mas sim esclarecimentos e ajustes padrão para alinhar 
com nossas práticas de negócio:

**QUESTÕES CRÍTICAS** (Requer Resolução):
1. [Listar issue crítica com descrição concisa]
2. [Issue crítica #2]

**QUESTÕES IMPORTANTES** (Requer Negociação):
3. [Issue alta com descrição]
4. [Issue alta #2]

**SUGESTÕES** (Melhorias Técnicas):
5. [Issue média com recomendação]
6. [Issue média #2]

Estou disponível para discutir e alinhar essas questões. Sugiro agendar 
ligação para [DATA/HORA] para que possamos resolver rapidamente.

Cordialmente,
[Seu Nome]
```

**Vantagens:**
- Categorizado (crítico/importante/sugestão) - mais fácil aceitar "sugestões"
- Profissional e respeitoso
- Abre diálogo (disponível para discussão)
- Não agressivo

### Template 2: Pushing Back on Unreasonable Clause

```
Assunto: Cláusula X - Proposta de Ajuste

Prezados [Nomes],

Após análise da cláusula X ("[específico texto]"), identificamos que 
excede padrões comerciais e cria exposição injustificada para ambas as partes.

**O Problema:**
[Explicar o impacto concreto - em linguagem acessível]

**Por que não é sustentável:**
- Viola princípio de equivalência (desproporção)
- Cria risco do qual ambos queremos nos proteger
- Mercado [mencionar prática comum] não aceita termos similares

**Proposta Alternativa:**
[Seu texto proposto que é razoável para ambas as partes]

**Benefício Mútuo:**
[Explicar como sua proposta protege ambas as partes]

Confiamos que essa reformulação seja aceitável. Ficamos abertos a sugestões 
alternativas que alcançem o objetivo de [reaseguração comum].

Cordialmente,
[Seu Nome]
```

**Estratégia:**
- Foco em "padrão mercado" (não opinião pessoal)
- Proposta concreta (não reclamação vaga)
- Benefício mútuo (não confrontacional)

### Template 3: Requesting Clarification

```
Assunto: Esclarecimento Necessário - Cláusula X

Prezados [Nomes],

Na releitura do contrato, identificamos que a Cláusula X é ambígua e precisa 
de esclarecimento para garantir alinhamento de expectativas.

**Texto Atual:**
"[Texto original]"

**Nossa Interpretação:**
[O que entendemos que significa]

**Questões Específicas:**
1. [Pergunta 1]
2. [Pergunta 2]

Para evitar mal entendidos depois, gostaríamos de confirmar que ambas 
as partes entendem isso de forma idêntica.

**Proposta para Clareza:**
[Propor linguagem mais específica]

Ficamos no aguardo da confirmação.

Cordialmente,
[Seu Nome]
```

**Vantagem:**
- Não é "problema seu" - é "ambíguo para todos"
- Propõe solução concreta
- Colaborativo (não adversarial)

## Negotiation Phrases

### Frases para Concordância Builder

| Situação | Frase |
|----------|-------|
| Iniciar negociação | "Apreciamos a proposta. Apenas gostaríamos de ajustar alguns pontos de clareza." |
| Discordar respeitosamente | "Entendemos a intenção, mas essa abordagem criaria [risco X]. Podemos explorar [alternativa]?" |
| Pedir concessão | "Essa é importante para nós. Podemos ceder em [Y] se ajustarmos [X]?" |
| Explicar limite legal | "Legislação brasileira não permite [situação]. Temos de reformular como [proposta]." |
| Reconhecer preocupação | "Entendemos sua preocupação com [risco]. Então propomos [salvaguarda]." |
| Encerrar discussão | "Chegamos a um ponto razoável. Vamos adiante com essa versão?" |

### Frases para Recusar Cláusula

| Situação | Frase |
|----------|-------|
| Muito arriscado | "Essa cláusula cria exposição [tipo] que não conseguimos aceitar. Alternativa razoável seria..." |
| Unilateral | "A obrigação é só nossa. Para equilibrar, sugerimos..." |
| Sem limite | "Sem cap em [X], expor-nos-emos indefinidamente. Limite de [R$ Y] seria justo." |
| Impossível cumprir | "Não conseguimos garantir [resultado]. Podemos comprometer com [esforço/meio]." |
| Contradiz lei | "Legislação brasileira (Art. X, CC) não permite isso. Reformularemos como..." |
| Padrão não-mercado | "Estudamos contratos similares. Ninguém aceita [termo]. Padrão é..." |

## Common Objections & Responses

### Objection 1: "Essas são nossas condições padrão"

**Response 1 (Soft):**
```
"Compreendemos que são seus padrões. Entretanto, em nossa experiência, 
[cláusula X] cria desalinhamento. Seria possível [ajuste] para refletir 
prática mais comum no mercado?"
```

**Response 2 (Firmer):**
```
"Padrões precisam ser justos para ambas as partes. [Cláusula X] é 
desfavorável. Consultamos [N] contratos similares, e nenhum inclui isso. 
Vamos reformular?"
```

---

### Objection 2: "Não temos flexibilidade nesse ponto"

**Response:**
```
"Compreendemos restrições internacionais. Qual é a razão desta cláusula? 
Se for [proteção contra X], podemos implementar [alternativa] que alcança 
o objetivo sem impor [risco Y]?"
```

**Estratégia:** Pergunte por quê. Frequentemente, a razão é legítima, 
mas a implementação é excessiva.

---

### Objection 3: "Outros clientes assinaram assim"

**Response:**
```
"Respeitamos que outros aceitaram. Porém, não é padrão no mercado, e 
cria risco inaceitável para nós. Gostaríamos de ser exceção (positiva) 
e fazer isso de forma justa para ambas as partes."
```

---

### Objection 4: "Isso é proteção nossa contra clientes ruins"

**Response:**
```
"Entendemos a preocupação. Podemos incluir [salvaguarda alternativa] 
que protege você contra [risco real] sem impor restrição geral? Por exemplo: 
[mecanismo específico]"
```

**Estratégia:** Reconhecer a preocupação, oferecer proteção alternativa.

## Concession Strategy

### When to Concede

| Situação | Ação |
|----------|------|
| Outro lado é inflexível | Conceda em ponto baixo para ganhar em ponto crítico |
| Benefício mútuo baixo | Solte se não impactar materialmente |
| Relação longa | Pequenas concessões agora = boa vontade futura |
| Ambíguidade menor | Melhor resolver agora do que litigar depois |

### When NOT to Concede

| Situação | Ação |
|----------|------|
| Risco crítico | Não ceda, negocie reformulação |
| Precedente ruim | Ceder agora complica futuros contratos |
| Lei proíbe | Nenhuma flexibilidade possível |
| Outro lado concede nada | Não seja o único cedendo |

### Trade-Off Framework

```
"Podemos aceitar [X] se você flexibilizar [Y]"

Exemplo: "Aceitamos limite indenizatório de 24 meses valores 
se excluir danos indiretos completamente"
```

## Red Flags in Negotiation

| Flag | Significado | Ação |
|------|-------------|------|
| Não respondendo | Evitação deliberada | Escalar ou considerar encerrar |
| Mudança de termos | Tentativa de renegociar | Fixar termos e pedir confirmação escrita |
| "Pequena mudança" (mas material) | Tática | Questionar: "Como isso muda risco?" |
| Pressão por prazo ("precisa assinar hoje") | Tática de urgência artificial | Não ceda a pressão; se crítico, melhor recusar |
| Pessoas mudando em reuniões | Falta de autoridade | Pedir confirmação por escrito de quem tem poder |

## Documentation Best Practices

### After Each Negotiation Round

```
EMAIL CONFIRMATÓRIO (Always Send):

Prezados [Nomes],

Confirmo que concordamos nos seguintes pontos na discussão de [DATA]:

✓ Ponto 1: [O que foi acordado]
✓ Ponto 2: [O que foi acordado]

Ainda em discussão:
□ Ponto 3: [Status]
□ Ponto 4: [Status]

Próximos passos:
- [Você] vai revisar cláusula X até [DATA]
- [Eles] vão enviar proposta alternativa até [DATA]
- Próxima reunião: [DATA/HORA]

Por favor confirme se discordar de qualquer ponto acima.

Cordialmente,
[Seu Nome]
```

**Benefício:** Cria registro de o que foi acordado oralmente. Previne 
"ela disse / ele disse" depois.

## When to Walk Away

**Consider rejecting contrato if:**

- ≥2 cláusulas CRÍTICAS sem flexibilidade
- Outro lado recusa QUALQUER negociação
- Lei brasileira seria violada
- Risco potencial > possível retorno
- De boa-fé não é aparente

**How to exit gracefully:**

```
"Após análise abrangente, identificamos que não conseguimos 
acomodar [questões fundamentais] neste momento. Talvez em 
contexto diferente no futuro. Agradecemos a oportunidade."
```

**Deixe porta aberta:** Você pode voltar, e eles podem ter 
novas informações/flexibilidade depois.
