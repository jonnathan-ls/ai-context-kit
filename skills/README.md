# Skills

Este diretório contém as definições de skills (habilidades) para o Antigravity.

## Estrutura

Organize seus arquivos de skills da seguinte forma:

```
skills/
├── skill-name/
│   ├── definition.json
│   ├── instructions.md
│   └── examples.md
└── README.md
```

## Como Adicionar uma Skill

1. Crie um novo diretório com o nome da skill
2. Adicione os arquivos de definição, instruções e exemplos
3. Documente a skill no arquivo README.md dentro do diretório da skill

## Exemplos

Consulte o diretório `examples/` para ver exemplos de skills.
