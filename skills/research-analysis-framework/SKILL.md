# Research & Analysis Framework

Evidence-first methodology for professional-grade investigation. Every claim is source-traceable. Every finding has a confidence level. Every recommendation flows logically from evidence.

## When to Use

- User needs to evaluate whether an idea, product, or project is viable
- User wants to research a topic with verifiable, structured output
- User needs a professional report with citations and confidence indicators
- User is conducting due diligence, market research, or feasibility analysis
- User needs contradictions between sources surfaced and resolved

## Core Principles

| Principle | Rule |
|-----------|------|
| **Evidence-First** | Every claim → traceable to a source or marked as assumption |
| **Transparent Confidence** | Every key finding has a confidence level |
| **Multidimensional** | No single score is sufficient — assess all relevant dimensions |
| **Contradiction Surfacing** | Never hide conflicting sources — document and resolve |
| **Assumption Documentation** | State what is being taken as given; show the risk if it breaks |

## Execution Flow

```
Phase 1 → SOURCE EVAL   : Tier and weight all sources (1-4)
Phase 2 → VIABILITY     : Score 6 dimensions 0-10 with evidence
Phase 3 → CONFIDENCE    : Assign confidence levels to all key findings
Phase 4 → TRIANGULATE   : Validate with multiple independent approaches
Phase 5 → REPORT        : Produce structured, audience-ready output
```

---

## Phase 1 — Source Evaluation Hierarchy

Categorize every source before using it as evidence:

```
TIER 1 — Primary, Authoritative
  Official standards, government regulations, primary research, permits

TIER 2 — Secondary, Established
  Professional guidelines, peer-reviewed analysis, industry benchmarks

TIER 3 — Tertiary, Contextual
  Analyst commentary, case studies, expert opinion (attributed)

TIER 4 — Supplementary, Anecdotal
  Forums, blogs, one-off examples — background only
```

When sources conflict:

| Conflict | Resolution |
|----------|-----------|
| Tier 1 vs Tier 2 | Tier 1 wins |
| Tier 2 vs Tier 3 | Tier 2 wins |
| Same tier | Assess recency and sample size |

## Phase 2 — Viability Assessment (6 Dimensions)

Score each dimension 0-10 with source evidence:

| Dimension | Key Questions |
|-----------|---------------|
| **Technical** | Can it be done? Does it meet standards? |
| **Financial** | Can it be afforded? What is the ROI? |
| **Regulatory** | Does it comply? What approvals are needed? |
| **Market** | Is there demand? What is the competitive position? |
| **Operational** | Can it be executed with available resources? |
| **Risk** | What are the failure modes and mitigations? |

Scoring scale:

```
9-10  EXCELLENT  — Strong alignment, high confidence, minimal risk
7-8   GOOD       — Solid foundation, few concerns, manageable risks
5-6   MODERATE   — Viable but requires attention, notable risks
3-4   WEAK       — Significant challenges, major mitigation needed
1-2   POOR       — Questionable viability, substantial barriers
0     BLOCKER    — Cannot proceed without fundamental change
```

---

## Phase 3 — Confidence Levels

Assign to every key finding:

```
🟢 HIGH (95%+)      — Multiple authoritative sources confirm, directly applicable
🟡 MEDIUM (70-94%)  — Primary + supporting evidence, clear pattern
🟠 LOW (50-69%)     — Limited evidence, contradictions, requires verification
🔴 UNVERIFIED       — No source support; label explicitly as assumption
```

Never present unverified claims as fact. Low-confidence findings must include a caveat.

---

## Phase 4 — Triangulation

Validate key findings using multiple independent approaches:

- **Source Triangulation** — Multiple independent sources agree → raises to HIGH
- **Methodological Triangulation** — Different analytical methods reach the same conclusion
- **Data Triangulation** — Different data types confirm the same pattern
- **Expert Triangulation** — Multiple domain experts independently align

Output: which findings are triangulated, which still lack corroboration, where evidence conflicts.

---

## Phase 5 — Report Structure

```
EXECUTIVE SUMMARY (1 page)
  — Key finding in one sentence
  — Recommendation
  — Viability score (0-10 composite, if applicable)
  — Top 3 risks
  — Timeline and next steps

DETAILED FINDINGS (per dimension)
  — Assessment approach
  — Source-based evidence
  — Key findings with confidence levels
  — Risks and mitigations

EVIDENCE & SOURCES
  — Source inventory with tier assignments
  — Contradictions identified and resolved
  — Confidence matrix for all major findings

RECOMMENDATIONS & NEXT STEPS
  — Priority actions
  — Decision criteria (what would change the recommendation?)
  — Contingency plans
  — Monitoring plan
```

## Audience Adaptation

| Audience | Format | Focus |
|----------|--------|-------|
| CEO/CFO | 1-page brief | Is it viable? What is the risk? |
| Director | 8-10 slide deck | What are we deciding? What is the data? |
| Technical Lead | Full report | How do we execute? What are the specs? |
| Finance Team | Cost model + tables | What does this cost? What is the ROI? |
| Project Manager | Implementation plan | What is the timeline? Resources needed? |

## Bias Mitigation Checklist

Before delivering any analysis:

- [ ] All claims are source-traceable or flagged as assumption
- [ ] Viability scored across ALL six dimensions
- [ ] Confidence levels assigned to major findings
- [ ] Contrary evidence addressed — not hidden
- [ ] Assumptions documented
- [ ] Limitations acknowledged
- [ ] Recommendations flow logically from findings
- [ ] No unqualified claims of certainty

## Anti-Patterns

- **Do not** present Tier 3 or Tier 4 sources as primary evidence.
- **Do not** skip dimensions where information is scarce — score them LOW or BLOCKER and explain why.
- **Do not** hide contradictions between sources — document and resolve them.
- **Do not** present unverified forecasts as fact — label them explicitly as assumptions.
- **Do not** omit confidence levels from key findings — they are not optional.
- **Do not** generate data that is not grounded in sources; fabricated figures destroy credibility.
