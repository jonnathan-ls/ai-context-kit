# Fair Example Skill

This skill helps analyze data and generate reports for users.

## Steps

Analyze the input and write the output report.

- Read the input file provided by the user.
- Check the data for obvious issues.
- Write the final summary.

The skill outputs a Markdown report with key findings. Make sure to include all relevant data.
