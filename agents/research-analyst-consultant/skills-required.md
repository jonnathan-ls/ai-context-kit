# Skills Required - research-analyst-consultant

## Primary Skill

### research-analysis-framework

This is the **foundational skill** that enables the agent to conduct professional-grade analysis.

| Component | Purpose |
|-----------|---------|
| **Source Evaluation Hierarchy** | Categorize and weight source credibility |
| **Viability Assessment Framework** | Score projects across 6 dimensions |
| **Metrics & KPIs** | Define and measure success indicators |
| **Accuracy & Confidence Levels** | Assign confidence to findings |
| **Research Triangulation** | Validate claims across methods |
| **Report Structure Templates** | Organize findings professionally |
| **Presentation Guidelines** | Adapt analysis for audiences |
| **Bias Mitigation Checklist** | Ensure quality & objectivity |

## Skill Activation

The `research-analysis-framework` skill is **automatically engaged** when the agent:

1. Reads and evaluates sources
2. Scores viability across dimensions
3. Assigns confidence levels
4. Structures reports
5. Adapts for stakeholders
