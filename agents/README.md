# Agents

Este diretório contém as definições de agentes para o Antigravity.

## Estrutura

Organize seus arquivos de agentes da seguinte forma:

```
agents/
├── agent-name/
│   ├── config.json
│   ├── instructions.md
│   ├── tools.md
│   └── README.md
└── README.md
```

## Como Adicionar um Agente

1. Crie um novo diretório com o nome do agente
2. Adicione as configurações, instruções e definições de ferramentas
3. Documente o agente no arquivo README.md dentro do diretório

## Exemplos

Consulte o diretório `examples/` para ver exemplos de agentes.
