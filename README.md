# My Antigravity Kit

Repositório para armazenar arquivos de contexto usados no Antigravity.

## 📋 Descrição

Este repositório serve como um kit de ferramentas e recursos para o Antigravity, organizando arquivos de contexto que incluem skills, workflows, agentes e outros componentes necessários para construir e gerenciar sistemas baseados em Antigravity.

## 📁 Estrutura do Repositório

```
my-antigravity-kit/
├── skills/          # Definições de skills (habilidades)
├── workflows/       # Configurações de workflows
├── agents/          # Definições de agentes
├── templates/       # Templates reutilizáveis
├── examples/        # Exemplos práticos
└── README.md        # Este arquivo
```

### Diretórios

- **`skills/`**: Contém as definições de skills que podem ser usadas no Antigravity
- **`workflows/`**: Armazena configurações de workflows e processos automatizados
- **`agents/`**: Define agentes personalizados com suas instruções e capacidades
- **`templates/`**: Templates prontos para criar novos componentes rapidamente
- **`examples/`**: Exemplos práticos para facilitar o aprendizado e uso

## 🚀 Como Usar

### Adicionando uma Nova Skill

1. Navegue até o diretório `skills/`
2. Crie um novo diretório com o nome da sua skill
3. Adicione os arquivos de definição e documentação
4. Consulte `skills/README.md` para mais detalhes

### Adicionando um Novo Workflow

1. Navegue até o diretório `workflows/`
2. Crie um novo diretório com o nome do seu workflow
3. Adicione a configuração e documentação
4. Consulte `workflows/README.md` para mais detalhes

### Adicionando um Novo Agente

1. Navegue até o diretório `agents/`
2. Crie um novo diretório com o nome do seu agente
3. Adicione as definições e instruções
4. Consulte `agents/README.md` para mais detalhes

## 📝 Convenções

- Use nomes descritivos e em kebab-case para diretórios (ex: `my-skill-name`)
- Sempre inclua um README.md em cada componente criado
- Mantenha a documentação atualizada
- Use os templates fornecidos como ponto de partida
